Програма Бухгалтерська звітність
Copyrigth (c) Сергій Лакас 1999-2003 Ужгород
Версія 1.0

Основні можливості програми:
• формування форми 1ДФ;
• Формування файла форми 1ДФ;
• Внесення змін до форми 1ДФ;
• Інтеграція з 1С6.0 (з можливістю повної адаптації під конфігурацію замовника);
• Інтеграція з 1С7.7 (з можливістю повної адаптації під конфігурацію замовника);


Системні вимоги:
• Pentium 166 і вище;
• 32 Mb RAM;
• Windows 9x, Windows 2000;
• 10 Mb вільного місця.

Демонстраційна версія розповсюджується безкоштовно,її можна завантажити з http:\\www.elikom.com
Демонстраційна версія відрізняється від зареєстрованої обмеженою функціональністю.
Після реєстрації вам буде вислано спеціальний ключ для активації програми.
