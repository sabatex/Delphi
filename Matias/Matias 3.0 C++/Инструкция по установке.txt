1.Проинсталировать Matias Server на сервере.
 (Перед установкой удалите старую версию сервера FireBird !!!)

2.Открить доступ для чтения пользователям базы к папке {setup}\ClientsShare.

3. Запускайте с этой папки для инсталяции клиентовнтов программу MatiasClient.exe
   (автоподстановка имени сервера и пути к базе) 
4. Установить МatiasAdmin.exe для администратора. 
  Введите пользователей в базу. Начальный логин и пароль администратора
  Login - SYSDBA
  Password - masterkey

!!! При изменении пароля для логина SYSDBA , сообщить Автору,
 иначе могут быть с автообновлением.
