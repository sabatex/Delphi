Порядок оплаты

Оплата осуществляется по безналичному расчету на основании счета-фактуры. Копия счета-фактура отправляется в виде письма по электронной почте. По запросу может быть также отправлен заказным письмом оригинал счета. Помимо этого потребуются следующие данные:

  • Ваш код ЄДРПОУ (для юридического лица) или идентификационный код (для физического);
  • Ваш адрес электронной почты - на него будет отправлен регистрационный код;
  • Название предприятия (или Ф.И.О. для физического лица);
  • Почтовый адрес, на который Вы получите комплект бухгалтерских документов.

Для плательщиков НДС также необходимо указать:

  • Индивидуальный налоговый номер (ИНН);
  • Номер свидетельства.
